## 🔍 F2P Pattern Explained

### What is F2P?
**F2P = Fail-to-Pass**

Tests fail at base commit, pass at head commit.

### Why F2P?
Proves the PR:
1. Adds new functionality
2. Has meaningful tests
3. Actually works

### How It Works

**Base Commit (main):**
```python
import notification_service  # ❌ ImportError
pytest test_notification_service.py  # ❌ FAILED
```

**Head Commit (PR):**
```python
import notification_service  # ✅ Success
pytest test_notification_service.py  # ✅ 18 PASSED
```
